# Python Calculator with CI/CD

A robust calculator implementation in Python with automated testing using GitHub Actions.

## Features

- Basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division

- Advanced mathematical operations:
  - Logarithm (base 10)
  - Square
  - Square root
